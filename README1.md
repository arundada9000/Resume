# Deploying your code as a webpage

- First you have to create a repository .

- You can drag and drop files or push using git.

- TO deploy your code as a webpage you must have index.html file in the root folder as it is default file for hosting used by github.

- Go to settings -> Pages -> Under Build and Deployement choose your branch and then save it.

- Wait some time and your code will be successfully hosted.
